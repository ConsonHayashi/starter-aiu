# 概览
```
- home 主页
- login 登陆
- logup 注册
- pass 找回密码
- user 用户信息
- privacy 隐私保护
- faq 常见问题
- agreement 用户服务协议
- about 关于我们
- notfound 找不到
- developing 正在开发
```

## 类型划分

- 用户基础设施 logup login pass user faq
- 客户基础设施 about home
- 人性化提示 notfound developing
- 政策文件 privacy agreement

## 需求

- 表单组件
- md 或 富文本 文件转译
